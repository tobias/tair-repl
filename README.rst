tair-repl
_________

A Simple REPL for taobao tair.

Usage
_____

::

    lein repl
    (use 'tair.repl)
    (put "key" "value)  ;; put something into tair
    (get "key")         ;; get something from tair
    (delete "key")      ;; delete something from tair
